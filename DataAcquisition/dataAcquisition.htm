SCADA Data Acquisition

All SCADA data changes are saved at the Web Server and applied in creating trending charts, graphs, and data tables for each SCADA process. Each SCADA process includes both its own Client Computer for interactive graphics and a Trending Computer using the database. This is an example using a method to accomplish this by saving data to a Windows Server in XML format via Javascript in .ASP (Classic ASP) files. (There are other options available to the vendor that can be applied to sending data to the server.) Each SCADA process includes its database that includes a data file for each device.
